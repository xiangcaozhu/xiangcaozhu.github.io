前端入门之前端自动化工具（Gulp）

发表于 2017-03-15 | 分类于 前端自动化

1、Gulp概述

将重复工作抽象成一个个的任务，机械化处理这些重复的任务。

2、Gulp四大接口

2.1获取文件源gulp.src(globs[, options]);

2.2转移文件到目标gulp.dest(path[, options]);

2.3监听文件变化gulp.watch(glob [, opts], tasks) 或 gulp.watch(glob [, opts, cb])

2.4添加执行任务gulp.task(name[, deps], fn);

3、Gulp常用插件

gulp插件黑名单

3.1公共插件

  • 自动加载各种插件,免require（）加载，调用时直接省略gulp-：gulp-load-plugins
  • 数据流的控制：pump
  • 重命名文件：gulp-rename
  • 映射源文件：gulp-sourcemaps
  • 合并css或js文件：gulp-concat
  • html版本添加：gulp-usemin
  • js&css文件自动增加md5后缀：gulp-dev
  • 清空文件：使用nodejs模块（del vinyl-paths）实现，gulp插件废弃了
  • 多浏览器静态服务：browser-sync

3.2处理html文件插件

  • 公共html文件合并 gulp-content-includer
  • 压缩html文件 gulp-htmlmin

3.3处理css文件插件

  • 编译 sass：gulp-sass

  • 编译 Less：gulp-less

  • 最小化 css 文件：gulp-clean-css

  • 自动添加前缀：gulp-autoprefixer

3.4处理js文件插件

  • gulp-babel转前提：babel-preset-es2015

  • es2016格式转es2015：gulp-babel

  • 最小化 js 文件：gulp-uglify

  • 校验 js 文件：gulp-jshint

3.5处理images图片文件插件

  • 最小化图像：gulp-imagemin
  • 最小化图像：imagemin-pngquant
