前端网页特效制作基础与实现原理

发表于 2017-06-03 | 分类于 JavaScript

概述

Js是用来控制html结构和css样式的行为前端脚本语言。

函数

函数：函数是由事件驱动的或者当它被调用时执行的可重复使用的代码块。一个函数实际上就是一个计算过程，计算完成之后的结果就是返回值。
arguments:对象的长度是由实参个数决定的，而不是形参个数。
return：所有的自定义函数默认没有返回值；

Math数学对象

  • Math.abs() 绝对值
  • Math.random() 随机数
  • Math.ceil() 向上取整
  • Math.floor() 向下取整
  • Math.round() 四舍五入函数

事件三要素

事件源（要触发的对象）.事件（怎么触发的这个事情） = function(事件对象event){ 事件处理函数 }

事件

  • click、mouseover、mousedown、mouseup、mouseenter、mousemove等
  • label模拟placeholder
    用户输入事件oninput（w3c）/onpropertychange （Ie 678）
    表单自动获得焦点:focus()
    选择表单内容:select();

事件对象

var event = window.event||event

  • event.stopPropagation()阻止冒泡w3c的方法
  • event.cancelBubble = true阻止冒泡IE的方法
  • event.screen:以电脑屏幕为基准的坐标
  • event.page:以网页页面为基准的坐标（ie不支持）
  • event.client:以浏览器可视区域为基准的坐标

数组与字符串常见操作

  • string.charAt(index)根据位置返回字符
  • string.indexOf(char)/string.lastIndexOf(char)根据字符返回位置
  • array.slice(startIndex,endIndex)/array.substring(startIndex,endIndex)从索引号0开始数startIndex/endIndex
  • array.substr(startIndex,count)从索引号0开始数startIndex后count个数
  • array.join(',')数据按照指定分隔符“,”拼接成字符串
  • string.split(',')字符串中按照指定的分隔符“,”组合成数据

dom的节点操作：

  • 获取html：document.documentElement
  • 获取head：document.head
  • 获取body：document.body
  • 获取标题：document.title
  • 新建节点：document.creatElement(tagName)
  • 插入节点：parentNode.appendChild(node)/insertBefore(插入的节点，参照节点)
  • 删除节点：parentNode.removeChild(node)
  • 克隆节点：cloneNode(boolean);深度拷贝boolean:true
  • 筛选节点：childNodes与nodeType类型值等于1-元素节点；2-属性节点；3-文本节点；
  • 获取节点属性：getAttribute（'attr'）
  • 设置节点属性：setAttribute ('attr','value')
  • 删除节点属性：removeAttribute('attr')

浏览器是否有DTD声明

标准模式是指，浏览器按W3C标准解析执行代码；怪异模式则是使用浏览器自己的方式解析执行代码，因为不同浏览器解析执行的方式不一样，所以我们称之为怪异模式。浏览器解析时到底使用标准模式还是怪异模式，与你网页中的DTD声明直接相关。

  • HTML5的文档类型声明：
    <!DOCTYPE html>
  • XHTML1.0严格模式的文档类型声明：
    <!DOCTYPE html PUBLIC “-//W3C//DTD XHTML 1.0 Strict//EN” “http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">

    判断是不是怪异模式的浏览器

    document.compatMode == "CSS1Compat"
    标准模式：CSS1Compat 已经声明
    怪异模式：BackCompat 未声明

    盒子模型的差异：

    IBM中国技术主题
    首先，两种模式可以利用box-sizing属性进行自行选择：
    标准模式：box-sizing:content-box;
    怪异模式：box-sizing:border-box;
    标准模式会被设置的padding撑开，而怪异模式则相当于将盒子的大小固定好，再将内容装入盒子。盒子的大小并不会被padding所撑开。
    标准模式：盒子总宽度/高度 = 内容区宽度 /高度+padding+border + margin。

    怪异模式：盒子总宽度/高度 = width/height + margin。

    检测屏幕宽度差异

    自己封装一个 返回可视区宽度和高度的函数。
    ie9及其以上的版本window.innerWidth
    标准模式：document.documentElement.clientWidth
    怪异模式：document.body.clientWidth

    offset家族


    一套方便的获取自身元素尺寸的办法
    offsetParent = 返回距离自身最近的有定位（absolute/relative/fixed）的父级元素dom对象
    offsetWidth = width + border + padding
    offsetHeight = Height + border + padding
    offsetLeft/offsetTop = 定位（absolute/relative/fixed）的父盒子border到子盒子自身border之间的距离。

offsetLeft/offsetTop自身可以不定位返回距离数字；而left/top必须自身要定位，同时要显示声明left/top值，返回行内样式left/top带单位的数字。
offsetParent返回有定位最近父级对象，parentNode返回就是亲父亲

scroll家族


scrollTop：滑动滚轮浏览网页的时候网页隐藏在屏幕上方的距离
scrollWidth:width + padding 超过内容的宽度，大小是内容的大小
scrollTo(xpos,ypos)方法可把内容滚动到指定的坐标。
xpos必需。要在窗口文档显示区左上角显示的文档的 x 坐标。
ypos必需。要在窗口文档显示区左上角显示的文档的 y 坐标。

body上滚动条

window.onscroll = function() { 页面滚动语句 }

局部滚动条

获取最外面包裹的wrap容器，在该容器的dom元素上获取scrollTop即可

父容器上要：overflow:scroll、height设置

模拟垂直滚动条

1.滚动条滑块的高度计算原理
容器的高度offsetHeight / 内容的高度pageY = 滚动条滑块的高度Y/容器的高度offsetHeight
Y= 容器的高度 / 内容的高度 容器的高度
2.滚动条滑块的移动距离对应内容的高度需要移动距离计算原理
（内容盒子高度 - 大盒子高度） / (大盒子高度 - 红色盒子的高度) 滚动条滑块的移动距离

client 家族

文档可视区域部分

clientWidth： width + padding不包含border
clientHeight： height + padding不包含border

获得鼠标选择内容（复制文本）

滚动底部：element.scrollHeight - element.scrollTop === element.clientHeight

动画原理

定时器:类似for循环，定时器最大的特点在于自动，可以设定时间。
匀速动画的原理：盒子本身的位置 + 步长
缓动动画的原理: 盒子本身的位置 + 步长 (不断变化的)
leader = leader + (target - leader) / 10

获取计算样式

//谁的/那个属性
function getStyle(obj,attr) {  
  if(obj.currentStyle){      
        // ie 等
      return obj.currentStyle[attr];  
  }else{
        // w3c 浏览器
      return window.getComputedStyle(obj,null)[attr];  
  }
}

匀速运动

function animate(function animate(obj,target){
    var speed = obj.offsetLeft < target ? 5 : -5;  // 用来判断 应该 +  还是 -
    obj.timer = setInterval(function() {
        var result = target - obj.offsetLeft; // 因为他们的差值不会超过5
        obj.style.left = obj.offsetLeft + speed + "px";
        if(Math.abs(result)<=5)  // 如果差值不小于 5 说明到位置了
        {
            clearInterval(obj.timer);
            obj.style.left = target + "px";  // 有5像素差距  我们直接跳转目标位置
         }
    },30)
}

缓动动画

function animate(obj,target){  function animate(obj,target){  //  第一个参数 动谁   第二个参数  动多少
   clearInterval(obj.timer);
   obj.timer = setInterval(function() {
         // 计算步长   动画的原理    盒子本身的位置  +  步长
         var step = (target - obj.offsetLeft) / 10;  // 步长
         step =  step > 0 ? Math.ceil(step) : Math.floor(step);  //  取整步长
         // obj.style.left = 盒子本身的位置  +  步长
         obj.style.left = obj.offsetLeft + step + "px";
         if(obj.offsetLeft == target){
              clearInterval(obj.timer);
          }
    },30)
}
