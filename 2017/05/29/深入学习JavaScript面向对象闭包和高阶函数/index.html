发表于 2017-05-29 | 分类于 JavaScript

1、闭包

闭包形成与变量作用域以及变量生命周期有关。
闭包最重要的一种作用：让局部变量不被销毁，方便下次再次调用，实现全局访问局部变量目的 。

1.1变量作用域

函数可以用来创建作用域，在函数声明的变量如果没有var声明，会沿着向上一直寻找变量声明，直到全局window对象上。
局部可以访问全局的，反之不可以。

1.2变量的生命周期

全局变量生命周期是永久的，局部变量会随着函数退出，自动销毁。在进行异步操作时，会经常存在变量过期的问题。

1.3闭包作用

提炼函数是代码重构的一种常见技巧。
如果函数不需要再其他地方使用，最好直接用闭包封闭起来。
